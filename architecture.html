CSDN MCP Server Architecture

AI Assistant → MCP Protocol → Playwright → CSDN Platform

Core Design

  • • Async Playwright with persistent browser context
  • • QR login page stays alive for callback
  • • Cookie-based session persistence
  • • Draft-first, publish requires user confirmation

Key Innovations

  • • Single browser context across all tool calls
  • • CodeMirror.setValue + keyboard events for auto-save
  • • Thread-safe async architecture
  • • WSL/headless compatible (no Chrome CDP needed)

Use Cases

  • • AI writes blogs → auto-publish to CSDN
  • • Content pipeline automation
  • • Multi-platform syndication (w/ WeChat)
  • • Scheduled content publishing (via Hermes cron)
